# IEC 62368-1 CB Test Report 英翻中（繁體）翻譯準則

本文件由你提供之 CB Test Report PDF 與對應的 AST-B 中文人工翻譯 Word 彙整而成，供後續 LLM 翻譯時遵循。

## 一、翻譯風格與格式規則（建議直接當作系統/開頭提示）

- 使用**繁體中文**；專有名詞優先採用本清單既有譯法。
- **條款/章節編號**（如 5.4.2.3.2.2、B.1.3、Y.6.2）與**標準代號**（IEC/EN/CNS、TRF No. 等）一律保留不改寫、不翻譯。
- **判定用語**：P→「符合」、F→「不符合」、N/A→「不適用」。若原表格為「⎯/—」或空白，維持空白或以原格式呈現（依版面需求）。
- 凡出現「See Annex X / See Table X / See appended table X」等參照語，統一譯為「(見附錄X)/(見表格X)」並保留括號與編號。
- 單位（V、A、Hz、°C、mm、kHz、MΩ…）與數值、倍率（×）、範圍符號（~、–、≤、≥）保持原樣；必要時僅調整中英文空格，不改變數值。
- 縮寫與等級符號（ES1/ES2/ES3、PS1/PS2/PS3、TS、RS、MS、B/S/R 等）保留；中文敘述可用既有譯法（例如「指示型安全防護」）輔助說明。
- 表格列內若有「.................................. :」等導引點線，中文多採**去除點線、保留冒號**的寫法（依既有範本）。
- 同一英文詞在不同條款若有既定譯法差異，以本清單的 **Preferred（最高頻）** 譯法為預設；必要時再依條款語境採用替代譯法。
